# robot-dynamics-control-

## Project Description
- 본 프로젝트는 matlab과 c 프로그래밍 언어를 이용하여 3개의 조인트를 탑재한 로봇 팔을 조작하는 프로젝트로, 폴더 내에 위치한 finalProject V2.c 에는 필요한 상수 및 변수와, 모터의 토크를 제어할 수 있도록 하는 함수들이 포함되어 있으며, 함수들이 매 1ms마다 호출이 되어 다양한 입력 값을 이용하여 지속적으로 로봇팔을 제어 및 조작할 수 있도록 합니다. 또한, matlab을 이용하여 이에 필요한 다양한 매트릭스의  그 값을 구하여 필요한 상수 값을 찾을 수 있습니다.
